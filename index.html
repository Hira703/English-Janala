What is the difference between var, let, and const?

var: Function-scoped, can be redeclared and updated.

let: Block-scoped, can be updated but not redeclared.

const: Block-scoped, cannot be updated or redeclared.

What is the difference between map(), forEach(), and filter()?

map(): Returns a new array after transforming each element.

forEach(): Iterates over elements but does not return a new array.

filter(): Returns a new array with elements that pass a condition.

Explain arrow functions and how they differ from regular functions.

Arrow Functions: A shorter syntax for functions using () => {}.

- They do not have their own this value.

- They cannot be used as constructors.

- Implicit return for one-liners ( const sum = (a, b) => a + b;).

How do JavaScript Promises work?

A Promise represents an operation that hasn't completed yet but is expected to.

- It can be Pending, Resolved, or Rejected.

- Use .then() for success and .catch() for errors.

Example:

let myPromise = new Promise((resolve, reject) => {
  setTimeout(() => resolve("Done!"), 2000);
});
myPromise.then(res => console.log(res));
How do closures work in JavaScript?

A closure is when a function remembers its outer variables even after execution.

Example:

function outer() {
  let count = 0;
  return function() { count++; console.log(count); }
}
const counter = outer();
counter(); // 1
counter(); // 2
